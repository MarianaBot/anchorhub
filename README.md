# AnchorahubBase - Proiect React Native
Structura fișierelor pentru proiectul tău multi-tenant.

## Instalare
1. Descarcă și instalează Node.js
2. Rulează `npm install`
3. Rulează `npx expo start`

## Structură
- `App.js`: Logica principală de interfață (Login + Dashboard)
- `branding.json`: Matricea de personalizare pentru multi-tenancy
- `database_schema.sql`: Fundația pentru baza de date (Supabase)
- `assets/`: Folderul pentru logouri și imagini
